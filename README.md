# CreatorFlow - Site Web Simple

Pages HTML simples pour Terms of Service et Privacy Policy.

## Fichiers

- `index.html` - Page d'accueil
- `terms.html` - Terms of Service
- `privacy.html` - Privacy Policy

## Tester localement

```bash
cd AMZ-MEM/projets/saaspro
python -m http.server 8000
```

Puis ouvrir: http://localhost:8000

## Déployer

1. Créer un repo GitHub
2. Copier les 3 fichiers HTML
3. Activer GitHub Pages
4. Utiliser les URLs pour TikTok Developer Portal
